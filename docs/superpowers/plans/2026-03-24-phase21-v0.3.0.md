**Context:**

## Task 2: CLI Service Lifecycle — Global Cancellation (Bug D3)

**Files:**
- Modify: `src/NexusMonitor.CLI/Program.cs`
- Modify: long-running command files (DashboardCommand, ProcessesCommand, PrometheusCommand, AlertsWatchCommand — check which ones exist)

**Context:**
`Spectre.Console.Cli 0.49.1` — `ICommandApp.RunAsync` does NOT accept a `CancellationToken`. Each long-running command (DashboardCommand, PrometheusCommand, etc.) creates its own local `CancellationTokenSource`. When CTRL+C is pressed, nothing cancels these local tokens.

**Fix:**
1. In `Program.cs`, declare a `static CancellationTokenSource GlobalCts` and wire it to `CancelKeyPress` and `ProcessExit`.
2. In each long-running command that creates its own local CTS, replace `new CancellationTokenSource()` with `CancellationTokenSource.CreateLinkedTokenSource(Program.GlobalCts.Token)`. This means the local token is cancelled when either the command itself decides to stop OR when CTRL+C fires.
3. For any command that polls (`while (!Console.KeyAvailable)`) without a CTS, add `&& !Program.GlobalCts.IsCancellationRequested` to the loop guard.

- [ ] **Step 1: Read Program.cs in full.**

  Identify where `services` is built and where `app.RunAsync(args)` is called.

- [ ] **Step 2: Add `GlobalCts` to Program.cs.**

  Before the `var app = ...` or service-build block, add:

  ```csharp
  // Global cancellation: CTRL+C or process exit cancels all long-running commands.
  internal static readonly CancellationTokenSource GlobalCts = new();
  ```

  Wire the event handlers immediately after (not in a method — must be before `RunAsync`):

  ```csharp
  Console.CancelKeyPress += (_, e) =>
  {
      e.Cancel = true;      // prevent abrupt termination; let commands clean up
      GlobalCts.Cancel();
  };
  AppDomain.CurrentDomain.ProcessExit += (_, _) => GlobalCts.Cancel();
  ```

- [ ] **Step 3: Find all long-running commands.**

  This means `cts.Token` is cancelled when either the global token fires (CTRL+C) or the command's own logic cancels it.
